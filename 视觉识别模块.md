# 视觉识别模块(已实现版)

现场人员识别 + 动作质量评估 + 实时日语语音互动。
机器人做操示范时,用它身上的外接摄像头看现场跟操的人,认出"谁"、
判断"做得好不好",在每节的语音空窗让机器人点名互动。

## 一句话原理

**机器人的播放时钟就是标准答案**——视觉不需要猜"大家在做什么",
只需要回答"该做的动作做到位没有"。判断交给视觉大模型(Qwen3-VL),
台词是固定集合,模型只输出结论,不编话。

## 硬件与数据流

```
[G1外接 罗技Brio 300 (PC2 USB)]
   │ brio_sender.py(自动部署到PC2,按设备名找相机,720p JPEG,TCP 5601)
   ▼
[笔记本 vision_coach.py 独立进程]
   │ ① 跟随播放时钟(vision_clock.json,由 demo_voice 每0.2s广播)
   │ ② ③④节各在第 2/4/5.5 拍抓3个关键帧(动作极值处)
   │ ③ 缩到640px发给 qwen3-vl-plus(国际版endpoint,实测约1.4s)
   │ ④ 编排出本节台词 → 原子写 observations.json
   ▼
[demo_voice.py(挂在仿真/真机执行循环上)]
     每节第 9~14 拍的语音空窗读取并播报(TTS忙顺延,窗关放弃)
```

两进程完全解耦:视觉崩了 = 语音读不到新结果 = 静默,演出不受影响。

## 认人

- 胸前贴 A4 大字母(A、B…),视觉大模型直接读字;
- 读不清时兜底:按画面从左到右 = A、B、C(现场约定站位别换);
- 人数由环境变量 `VISION_NAMES` 配置(当前 `AB` 两人,默认 `ABC`);
- 只评"正在跟操的人",旁观/坐着/路过的不评,拿不准宁可不说。

## 评什么(只评③④,简单直观、误判率最低)

| 节 | 评判标准(说给模型听) | 不达标台词 |
|---|---|---|
| ③ 腕回し | 圈够大吗?手过头顶没有? | 腕をもっと大きく回しましょう |
| ④ 胸反らし | 双臂展开没有?举过肩没有? | 胸を大きく開きましょう / 腕をもっと高く上げましょう |

三档反馈(每节最多点名1人,轮换主角 ③A ④B,④结束补漏保证人人被点到):

1. **纠正**(明确不达标):〇〇さん、腕をもっと大きく回しましょう
2. **打气**(达标但蔫,energy=low):〇〇さん、もっとテンション上げていきましょう!
3. **表扬**(达标又带劲):よく頑張りました!/ いい感じですね!その調子!/ とても上手です!/ 元気いっぱいですね!

原则:**纠正要有把握,表扬可以宽松**——说错表扬没事,说错批评很尴尬。
所有"名字×台词"组合开演前全部 TTS 预取,到点零等待。

## 可靠性设计(真机踩坑换来的)

- **早发起**:第5.5拍采完即送评,给网络抖动留5秒余量(曾遇到单次15s的延迟尖峰)
- **旧不挡新**:评判不加互斥锁,迟到的旧结果直接作废丢弃,绝不覆盖新节的结果
- **演出中静默预览**:调试标注(画框认人)只在空闲时刷新,演出中不抢带宽
- **设备号按名找**:Brio 的 /dev/videoN 每次重启会变,发送端按设备名自动定位
- **绝不自动开笔记本摄像头**:机器人相机连不上就停下提示,只有显式 `--cam 0` 才用本机

## 文件清单

| 文件 | 作用 |
|---|---|
| `vision_coach.py` | 主程序:取流、时钟同步、采样、VLM评判、反馈编排 |
| `vision_rules.py` | 纯数据:评判标准、台词集、关键帧拍号、主角轮换(现场改这里) |
| `brio_sender.py` | PC2端推流(自动部署,一般不手动碰) |
| `robot_camera.py` | 相机取流库 + 独立预览工具(--probe/--source head) |
| `observations.json` / `vision_clock.json` | 视觉↔语音的两个通信文件 |

## 运行

```powershell
# 视觉(独立窗口;--show 带画面和认人标注)
$env:VISION_NAMES="AB"
.\.venv-real\Scripts\python.exe vision_coach.py --show

# 可选:--cam 0 用笔记本摄像头;--standalone 无时钟手动对齐
```

调试窗口:绿框=认出字母,黄框?=看到人没认出字,左上 `seen: A,B` 一目了然;
演出时状态栏会显示"采样1/3→评判中",第9~14拍机器人开口即闭环成功。

## 模型

- 评判:`qwen3-vl-plus`(env `VISION_MODEL` 可换,备选 qwen3-vl-flash 更快/235b 更强)
- 预览:`qwen3-vl-flash`(env `VISION_PREVIEW_MODEL`)
- 全部走 dashscope 国际版 endpoint,base64 传图,640px/质量70
